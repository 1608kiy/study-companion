# 智学伴 - AI学习陪伴平台

> 记录努力，看见进步，AI陪伴，心里踏实

## 项目简介

智学伴是一个专注于考公考编的学习陪伴工具，通过记录学习时间、写日记、AI分析，帮助用户看见自己的努力，心里踏实，更有底气。

## 核心功能

- **番茄钟计时**：精确记录每次学习的时长
- **学习打卡**：每日打卡，记录连续学习天数
- **强制日记**：写完日记才能打卡成功，促进深度反思
- **AI智能分析**：根据学习数据给出个性化建议
- **数据可视化**：月历视图、统计图表，让努力看得见

## 技术栈

| 层级 | 技术 |
|------|------|
| 后端 | Spring Boot 3 + MyBatis-Plus |
| 前端 | Vue3 + Element Plus |
| 小程序 | Uniapp |
| 数据库 | MySQL + Redis |
| AI | 通义千问/GLM API |

## 快速开始

### 后端启动

```bash
cd backend
mvn spring-boot:run
```

### 前端启动

```bash
cd frontend
npm install
npm run dev
```
