# 智学伴 - 项目概述

## 一、项目名称

**智学伴** - AI学习陪伴平台

## 二、项目背景

考公考编备考是一个漫长且需要高度自律的过程。很多考生在备考过程中遇到以下问题：

1. **时间管理困难**：不知道自己每天学了多久，时间都去哪了
2. **缺乏正反馈**：看不到自己的进步，容易焦虑和放弃
3. **难以坚持**：没有监督和陪伴，容易懈怠
4. **反思不足**：每天学完就忘了，没有深度反思

市面上有很多学习工具，但大多功能繁杂，不够纯粹。我们需要一个**简洁、专注、有温度**的学习陪伴工具。

## 三、目标用户

- 公务员考试备考人员
- 事业编制考试备考人员
- 其他需要长时间学习的用户

## 四、核心价值

### 1. 记录努力，看见进步
- 精确记录每次学习的时长、科目
- 可视化展示学习数据
- 让努力看得见，心里踏实

### 2. AI陪伴，智能建议
- AI根据学习数据给出个性化建议
- 定期生成学习报告
- 智能判断专注度、补签等

### 3. 强制日记，深度反思
- 每天写日记才能打卡成功
- 促进深度反思，总结经验教训
- AI辅助生成日记

### 4. 数据驱动，科学备考
- 分析学习规律
- 识别薄弱科目
- 优化学习计划

## 五、项目定位

**定位**：一个专注于考公考编的学习陪伴工具

**特点**：
- ✅ 简洁纯粹：只做学习记录和陪伴
- ✅ AI加持：智能分析、建议、日记生成
- ✅ 温暖治愈：界面风格温暖，有陪伴感
- ✅ 数据可视化：让努力看得见

**不是什么**：
- ❌ 不是题库刷题工具
- ❌ 不是社交平台
- ❌ 不是复杂的项目管理系统

## 六、项目目标

### 短期目标（毕业设计）
- 完成核心功能开发
- 实现Web端+小程序
- 部署上线，自己能用

### 长期目标（可选）
- 优化用户体验
- 增加更多AI功能
- 考虑商业化

## 七、技术栈概览

| 层级 | 技术 |
|------|------|
| 后端 | Spring Boot 3 + MyBatis-Plus |
| 前端 | Vue3 + Element Plus |
| 小程序 | Uniapp |
| 数据库 | MySQL + Redis |
| AI | 通义千问/GLM API |
