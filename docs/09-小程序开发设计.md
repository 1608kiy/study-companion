# 智学伴 - 微信小程序开发设计文档

## 一、文档概述

### 1.1 文档目的
本文档是智学伴微信小程序端的详细设计文档，用于指导开发人员完成小程序端的实现。

### 1.2 项目背景
智学伴是一个考公考编学习陪伴平台，目前已完成 Web 端（Vue3）和后端（Spring Boot）的开发。本文档描述微信小程序端的设计方案，小程序与 Web 端共用同一后端服务，数据完全互通。

### 1.3 术语定义
| 术语 | 说明 |
|------|------|
| Uniapp | 跨平台框架，支持编译到微信小程序 |
| TabBar | 小程序底部导航栏 |
| WXML | 微信小程序模板语言 |
| WXSS | 微信小程序样式语言 |

### 2.3 与 Web 端的关系
- 同一用户在 Web 端和小程序端使用相同账号登录
- 学习记录、打卡数据、日记等完全同步
- 计时器状态通过后端 Redis 同步
- 用户设置通过后端同步

## 四、API 接口复用

### 4.1 接口规范
- 基础路径：`/api/v1`
- 认证方式：JWT Token
- 响应格式：`{ code: 200, message: "success", data: T }`

### 4.2 接口清单
| 模块 | 端点 | 方法 | 说明 |
|------|------|------|------|
| 认证 | `/auth/login` | POST | 登录 |
| 认证 | `/auth/register` | POST | 注册 |
| 用户 | `/user/profile` | GET/PUT | 个人信息 |
| 科目 | `/subjects` | GET/POST | 科目管理 |
| 计时 | `/study-records/timer/*` | POST/GET | 计时器 |
| 记录 | `/study-records` | GET | 记录列表 |
| 记录 | `/study-records/paged` | GET | 分页记录 |
| 记录 | `/study-records/stats` | GET | 学习统计 |
| 打卡 | `/check-in/today` | GET | 打卡状态 |
| 打卡 | `/check-in` | POST | 打卡 |
| 日记 | `/diaries` | GET/POST | 日记管理 |
| 日记 | `/diaries/paged` | GET | 分页日记 |
| 日记 | `/diaries/generate` | POST | AI生成 |
| 目标 | `/goals` | GET/POST | 目标管理 |
| 目标 | `/goals/stats/*` | GET | 统计 |
| AI | `/ai/chat` | POST | AI对话 |
| AI | `/ai/weekly-report` | POST | 周报 |
| AI | `/ai/monthly-report` | POST | 月报 |

## 五、页面设计

### 5.1 路由配置
| 路径 | 页面 | 类型 | 登录 |
|------|------|------|------|
| `/pages/home/home` | 首页 | Tab | 是 |
| `/pages/study/study` | 学习 | Tab | 是 |
| `/pages/diary/diary` | 日记 | Tab | 是 |
| `/pages/stats/stats` | 统计 | Tab | 是 |
| `/pages/ai/ai` | AI助手 | Tab | 是 |
| `/pages/settings/settings` | 设置 | 子页 | 是 |
| `/pages/login/login` | 登录 | 子页 | 否 |
| `/pages/register/register` | 注册 | 子页 | 否 |

### 5.2 各页面功能

**首页**：统计卡片、趋势图、打卡
**学习**：计时器、科目选择、记录列表
**日记**：日历、详情、AI生成
**统计**：图表、热力图
**AI**：对话、周报/月报
**设置**：个人信息、目标、退出

## 六、数据流设计

### 6.1 认证流程
```
用户输入 → authApi.login() → token
  → uni.setStorageSync → switchTab
```

### 6.2 请求拦截
```
API调用 → 添加token → uni.request
  → 401清除token → 200返回data
```

### 6.3 计时器同步
```
Web端计时 → Redis存储 → 小程序端恢复
```

## 八、注意事项

### 8.1 小程序限制
- 网络请求用 `uni.request`
- 本地存储用 `uni.setStorageSync`
- 路由用 `pages.json`
- 主包限制 2MB

### 8.2 与Web端差异
| 功能 | Web端 | 小程序端 |
|------|-------|----------|
| UI框架 | Element Plus | uni-ui |
| 图表 | ECharts | echarts-for-weixin |
| Markdown | DOMPurify | rich-text |
