# 智学伴 - 前端设计

## 一、界面风格

### 1.1 设计理念
- **温暖治愈**：使用暖色调，给人陪伴感
- **简洁纯粹**：功能清晰，操作简单
- **数据可视化**：让努力看得见

### 1.2 配色方案
| 颜色 | 色值 | 用途 |
|------|------|------|
| 主色 | #FF6B6B | 温暖的红色，用于按钮、强调 |
| 辅助色 | #4ECDC4 | 清新的青色，用于图表、进度 |
| 背景色 | #FFF9F9 | 浅粉色背景 |
| 文字色 | #333333 | 主要文字 |
| 次要文字 | #999999 | 次要信息 |

### 1.3 深色模式
- 背景色：#1A1A1A
- 文字色：#FFFFFF
- 主色保持不变

## 五、响应式设计

### 5.1 断点设计
| 断点 | 宽度 | 设备 |
|------|------|------|
| xs | <576px | 手机 |
| sm | ≥576px | 大手机 |
| md | ≥768px | 平板 |
| lg | ≥992px | 小桌面 |
| xl | ≥1200px | 桌面 |
| xxl | ≥1600px | 大桌面 |

### 5.2 布局适配

- 平板：双列布局
